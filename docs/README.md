# Documentação do projeto STM32F411RE Internal Temperature to UART

Este diretório concentra toda a documentação do projeto.

## Índice

- `REQUIREMENTS.md` – requisitos funcionais e não funcionais.
- `PLAN.md` – planejamento, roadmap e riscos.
- `CHANGELOG.md` – histórico de mudanças.
- `architecture.md` – arquitetura do firmware e matemática do sensor de temperatura.
- `how-to/` – tutoriais passo a passo (build, testes, deploy, troubleshooting).
- `guidelines/` – padrões de código, logging e segurança.
- `integrations/` – integrações externas (quando existirem).

Toda a documentação é escrita em **português brasileiro**, com foco didático.
