# Changelog

Formato de versionamento: `x.y.z`

- `x` – mudanças de paradigma / quebra de compatibilidade.
- `y` – mudanças estruturais compatíveis.
- `z` – `z` par → novos recursos; `z` ímpar → correções de bugs.

## [0.1.0] - 2025-12-10

### Added

- Projeto base STM32F411RE com CubeMX + CMake.
- Leitura do sensor interno de temperatura e envio pela USART2.
- Documentação inicial em `docs/`.
