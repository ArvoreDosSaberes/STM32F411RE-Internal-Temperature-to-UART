# Diretrizes de segurança

- Não expor chaves, senhas ou dados sensíveis no código-fonte.
- Este projeto é didático e não lida com dados confidenciais, mas as boas
  práticas devem ser mantidas.
- Ao integrar com outros sistemas (cloud, APIs), registrar variáveis de
  ambiente e segredos apenas em documentação apropriada, nunca em código.
